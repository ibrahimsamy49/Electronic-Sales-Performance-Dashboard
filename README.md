# Electronic Sales Performance Dashboard

## Project Overview
Interactive Power BI dashboard analyzing electronic sales performance across 24 months (2023–2024) and 7 product categories including Tablets, Laptops, Smartphones, Cameras, Gaming, Headphones, and TV.

## Key Metrics
- Total Revenue: $51M
- Total Target Revenue: $49M
- Overall Achievement: 104%
- Highest Monthly Revenue: $4.46M
- Months Exceeding Target: 18 out of 24

## Dashboard Pages
- **Revenue Overview** — Total revenue vs target, monthly trends, achievement % by month
- **Category Analysis** — Performance breakdown across 7 product categories
- **Detailed Data Table** — Month-by-month revenue, target, and gap analysis

## Key Insights
- Tablets generated the highest revenue among all 7 categories
- Q4 showed peak performance with November as the strongest month
- Revenue exceeded targets in 18 out of 24 months
- Overall achievement reached 104% across the full 2-year period

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX
- Data Modeling
